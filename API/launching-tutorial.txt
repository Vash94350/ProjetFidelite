Pour pouvoir utiliser l'API,
-> vous devez créer les trois bases de données indiquées dans le config.json (dossier : config)
    "database_development_fid", "database_test_fid", "database_production_fid"
-> votre server de bdd doit contenir un compte root avec null en mot de passe (si différent, le changer pour vous dans le config.json)
-> une fois celles-ci créées, mettez-vous dans le dossier de l'API
-> vous pouvez lancer l'API avec un "npm start", les base de données vont se créer automatiquement
-> une fois celles-ci créées, n'oubliez pas d'enlever le { force: true } au sein du app.js (sinon cela effacera la bdd à chaque fois)
