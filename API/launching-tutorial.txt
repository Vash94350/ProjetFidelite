Sur windows :

-> Allumer le serveur mysql (xampp par exemple)
-> Cliquer sur le _server-launcher.bat
-> Cela créer les trois bases de données
-> Cela lance npm install et npm start dans le dossier API
-> une fois cela fait, n'oubliez pas d'enlever le { force: true } au sein du app.js (sinon cela effacera les tables à chaque fois)

Sur tout autre support (ou si batch ne fonctionne pas) :

Pour pouvoir utiliser l'API,
-> vous devez créer les trois bases de données indiquées dans le config.json (dossier : config)
    "database_development_fid", "database_test_fid", "database_production_fid"
-> votre server de bdd doit contenir un compte root avec null en mot de passe (si différent, le changer pour vous dans le config.json)
-> une fois celles-ci créées, mettez-vous dans le dossier de l'API
-> vous pouvez lancer l'API avec un "npm start", les tables vont se créer automatiquement
-> une fois celles-ci créées, n'oubliez pas d'enlever le { force: true } au sein du app.js (sinon cela effacera les tables à chaque fois)
